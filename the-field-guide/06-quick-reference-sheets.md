# Quick Reference Sheets

Condensed versions of the curriculum's key frameworks, checklists, and sequences. Designed to be printed, bookmarked, or pinned to the wall.

## 1. Universal Decision Framework — The Six Questions

```
1. Am I safe?
   → If no: protect neck, retract arms, build frames, recover alignment
   → If yes: proceed to #2

2. Who has inside position?
   → If opponent: pummel, re-frame, reclaim before attacking
   → If me: maintain and proceed to #3

3. Who controls head and hips?
   → If opponent: fight for head position, free hips with movement
   → If me: tighten control and proceed to #4

4. Who has mobility?
   → If I'm stuck: create space with frames, free hips first
   → If I can move: proceed to #5

5. What is the next best position?
   → Advance one tier on the hierarchy — never skip tiers
   → Tier 1 → Tier 2 → Tier 3 → Tier 4 → Tier 5 → Tier 6

6. What threats exist right now?
   → Scan: opponent threats (defend) + my opportunities (attack)
   → Most immediate threat determines next action
```

## 2. Positional Hierarchy

```
WORST                                               BEST
  │                                                   │
  ▼                                                   ▼

Tier 1: DEFENSIVE              Tier 5: DOMINANT
  Back controlled (bottom)       Side control (top)
  Mount (bottom)                 Mount (top)
  Side control (bottom)          Back control (top)

Tier 2: RECOVERY               Tier 6: SUBMISSION
  Turtle                         Strangles
  Half guard (bottom)            Joint locks
  Transitional frames

Tier 3: NEUTRAL / GUARD
  Closed guard (bottom)
  Open guard variations

Tier 4: OFFENSIVE
  Passing positions
  Top half guard

→ RULE: Advance one tier at a time. Never skip.
```

## 3. Core Mechanical Concepts — Quick List

| Mechanic              | One-Line Definition                                        |
| --------------------- | ---------------------------------------------------------- |
| Base                  | Weight distribution that creates stability                 |
| Posture               | Spinal alignment that creates structural integrity         |
| Frames                | Rigid skeletal structures that maintain distance           |
| Wedges                | Body parts inserted into gaps to block movement            |
| Inside Position       | Limbs closer to opponent's centerline than theirs to yours |
| Head Control          | Controlling the head to dictate body direction             |
| Hip Control           | Controlling the hips to limit movement and power           |
| Connection/Separation | Managing body contact — close to control, open to escape   |
| Pressure/Mobility     | Weight to immobilize vs speed to reposition                |

## 4. Level 2 Escape Sequence

```
Step 1: SURVIVE
  → Protect neck, tuck chin, tighten elbows, do not panic

Step 2: ALIGN
  → Recover spinal alignment, get to one hip, face the opponent

Step 3: FRAME
  → Build rigid frames on neck/hip/bicep — bone, not muscle

Step 4: HAND FIGHT
  → Strip threatening grips before they establish control

Step 5: ELBOW-KNEE
  → Connect elbow to knee to create the guard recovery barrier

Step 6: RECOVER
  → Insert knee, re-establish guard, begin offense
```

---

## 5. Level 5 Passing Gate Sequence

```
Gate 1: FEET
  → Beat the feet — strip grips, control ankles, pass the feet line

Gate 2: KNEES
  → Beat the knees — clear knee shields, collapse butterfly hooks

Gate 3: HIPS
  → Beat the hips — flatten hips to mat, deny hip escape

Gate 4: UPPER BODY
  → Clear frames — remove forearm frames, establish crossface

→ COMPLETE: Arrive in side control with control established
→ RULE: Work the gates in order. Do not skip.
```

## 6. Level 6 Submission Sequence — Five Phases

```
Phase 1: ISOLATE
  → Separate the target limb or neck from the opponent's defensive structure

Phase 2: EXPOSE
  → Create the angle and access needed for the submission

Phase 3: BREAK DEFENSE
  → Remove grips, frames, and defensive posture protecting the target

Phase 4: CONTROL
  → Maintain positional control throughout the finishing sequence

Phase 5: FINISH
  → Apply the submission with mechanical efficiency — hips, not hands
```

## 7. Level 7 System Integration — The Match Loop

```
1. Establish position (L3/L4/L5)
    ↓
2. Scan for opportunity (Anticipation + Transition Awareness)
    ↓
3. Choose the pace (Pacing + Efficiency)
    ↓
4. Present the dilemma (Dilemma Creation)
    ↓
5. Chain through defenses (Chaining Attacks)
    ↓
6. Anticipate and pre-position (Anticipating Reactions)
    ↓
7. Exploit transitions (Transition Awareness)
    ↓
8. Recover and repeat (Pacing + Efficiency)
    ↓
→ Return to step 2 — the cycle continues
```

## 8. Decision-Making Checklist — Full Match

| Check        | Question                                | If No...                                  |
| ------------ | --------------------------------------- | ----------------------------------------- |
| Safety       | Am I safe?                              | Survive first — frame, protect, structure |
| Position     | What position am I in?                  | Identify tier and opportunities           |
| Opportunity  | Is there a clear opportunity right now? | Hold and scan — do not force              |
| Dilemma      | Can I present two threats?              | Establish better control first            |
| Chain        | What is the next link?                  | Always have the next attack prepared      |
| Anticipation | What will they do?                      | Pre-position for most likely defense      |
| Energy       | Am I spending efficiently?              | Find the structural/gravity alternative   |
| Pace         | Should this moment be fast or slow?     | Burst if high opportunity, recover if low |

## 10. Dilemma Creation Template

```
Position: [current position]
Threat A: [first attack option]
Threat B: [second attack option]

If opponent defends A → B opens because: [reason]
If opponent defends B → A opens because: [reason]
If opponent defends neither → finish with [better option]

Preparation needed:
  → [grip/position/angle required to make both threats real]
```

Use this template to design and document your own dilemmas from any position.
