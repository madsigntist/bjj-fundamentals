## Phase 2: Print Publication (Lulu)

### Manuscript Prep

- [ ] Define final page size (6x9 standard or 7x10 trade)
- [ ] Export all Markdown content to a single compiled document
- [ ] Format for print: set fonts, heading sizes, margins, line spacing
- [ ] Handle images and diagrams (print-safe resolution, 300 DPI)
- [ ] Add front matter: title page, copyright page, dedication, TOC
- [ ] Add back matter: about the author, index (if needed)
- [ ] Export final manuscript as PDF (print-ready)

### Lulu Paperback Setup

- [ ] Create Lulu account (lulu.com)
- [ ] Upload interior PDF
- [ ] Design or commission cover (Lulu cover wizard or custom)
- [ ] Set trim size, paper type (cream vs. white), and binding
- [ ] Order proof copy and review print quality
- [ ] Make any print corrections
- [ ] Publish paperback and set price
- [ ] Set up distribution (Lulu marketplace, Amazon, Ingram if applicable)

### Lulu eBook Setup

- [ ] Format manuscript for ebook (EPUB-friendly structure, no fixed layouts)
- [ ] Export as EPUB or let Lulu convert from DOCX/PDF
- [ ] Validate EPUB file (use EPUBCheck or Lulu's built-in validator)
- [ ] Upload ebook to Lulu
- [ ] Set ebook price and distribution channels
- [ ] Test ebook on multiple readers (Kindle app, Apple Books, Google Play)
